# 🧠 Smart Online Examination System

## 📌 Overview

A full-stack web application that allows students to take online tests, view results, and analyze their performance.

Built using Flask, JavaScript, and SQLite, this project simulates a real-world online examination platform with authentication, dynamic question rendering, and result analytics.

---

## 🚀 Features

* 🔐 User Registration & Login (Session-based authentication)
* 📝 Dynamic Quiz Generation using JavaScript
* 🎯 Real-time Score Calculation
* 📊 Performance Visualization using Chart.js
* 🗄️ SQLite Database for storing users and results
* 🔒 Protected Routes (No access without login)
* 🔁 Retake Test functionality

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/online-examination-system.git
cd online-examination-system
```

### 2. Install dependencies

```
pip install flask
```

### 3. Run the application

```
python app.py
```

### 4. Open in browser

```
http://127.0.0.1:5000/
```
